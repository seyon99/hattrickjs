drawNPlayerPassWithCoordsCustom(playName, backgroundImgLink, width, height, passArray, playersArray)

Draws a pass between N players on a canvas with a custom background image. Can pass in canvasId in place of playName here if the canvas exists already and you just want to draw a play on top of it. If the user chooses to use the function in this way, the width, height, and backgroundImgLink must be the same as the old canvas. Otherwise, a new play can also be generated by passing in a unique playName as the first parameter.
Parameters:
Name Type Description
playName * name of play
backgroundImgLink * link to a background image to fill the canvas with
width * width of canvas
height * height of canvas
passArray * array of coordinates representing player locations on the canvas in the format: ["x1,y1", "x2,y2", ...], with the elements being ordered in the order of the ball movement
playersArray * array of player objects (must include players involved in the passing scheme)
