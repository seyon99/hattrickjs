Hattrick.js

About

Hattrick.js is a frontend JavaScript library used for visualizing soccer plays using data/coordinate points. Its primary uses are to visualize a soccer play (i.e. a specific set of passes between players) and the details of that play. There is also an option to generate scouting reports for players being visualized.

Getting Started

In order to use Hattrick.js, users must create a script tag in the head of their HTML file like so:

If the user has the hattrick.js file containing the API saved locally:
<script type="text/javascript" src="hattrick.js"></script>
Otherwise, the following code can be used in the head:
<script type="text/javascript" src="https://hattrickjs.herokuapp.com/hattrick.js"></script>

Inside the HTML body of any file where the script above was added in the HTML head, the following code can be used within script tags to instantiate a Hattrick object so that the API methods can be called. Alternatively, users can include a reference to Hattrick.js in external javascript files.

const hattrick = new Hattrick(); hattrick.newBlankPlayCanvas("Ted Lasso Scoring Special", "#38b000", 1000, 550);

In the above snippet, we made a call to the newBlankPlayCanvas method, but the user may call any of the API methods using the hattrick object.

Drawing Up Plays On Canvas

Using Generic Markers to Represent Players

Code:

const hat = new Hattrick(); hat.newBlankPlayCanvas("Ted Lasso Scoring Special", "#38b000", 1000, 550); coordsArray = ["50,30", "200,120", "150,300", "500,200", "920,200", "700,500"]; hat.drawPlayerUsingCoordsOnly("TedLassoScoringSpecial", coordsArray); passArray = ["50,30", "200,120","700,500","150,300", "500,200","7,265"]; hat.drawNPlayerPassWithCoords("TedLassoScoringSpecial", passArray);

Using Image Markers to Represent Players

Code:

const hat = new Hattrick(); const soccerBackground = "https://upload.wikimedia.org/wikipedia/commons/thumb/b/b9/Soccer_field_-_empty.svg/2560px-Soccer_field_-_empty.svg.png" hat.newCustomPlayCanvas("Play with custom background", soccerBackground, 1000, 550); const pnames = ["Mbappe", "Messi", "Neymar"]; const imgs = ["img1.jpg", "img2.jpg", "img3.jpg" ]; const tname = "Paris Saint-Germain F.C."; const heights = [1.83, 2.03, 1.72]; const gpgs = [2.1, 3.4, 3.0]; const apgs = [4.2, 4.5, 2.4]; const sogs = [15, 12, 11]; const pCoordsArr = ["110,50", "400,200", "200,300"]; const playerArray = hat.generatePlayerStructArray(pnames, imgs, tname, heights, gpgs, apgs, sogs, pCoordsArr); hat.drawPlayersUsingObjects("Playwithcustombackground", playerArray); hat.drawNPlayerPassWithCoordsCustom("Playwithcustombackground", soccerBackground, 1000, 550, pCoordsArr, playerArray);
